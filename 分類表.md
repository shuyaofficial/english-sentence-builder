# 品詞 × 意味グループ 分類表

英会話で「言いたいこと」を最短で英語に変換するため、コア語彙を
**品詞 → 意味グループ → サブグループ** に細分化する際の定義集。
分類は思いつきではなく、確立した言語学の枠組みに接地している。

- **動詞** … CEFR の言語機能カテゴリ（意見/依頼/提案/同意…）＋ Levin の意味的動詞クラス（移動・状態変化・伝達・授受・知覚/認知）
- **名詞** … COCA/NGSL の意味フィールド（時間・人・場所・抽象プレースホルダー…）
- **形容詞** … Dixon の意味型（DIMENSION/AGE/VALUE/COLOUR/PHYSICAL PROPERTY/HUMAN PROPENSITY/SPEED/DIFFICULTY/SIMILARITY/QUANTIFICATION/POSITION）
- **副詞** … COCA の意味的分類（程度・頻度・時・場所・様態・談話標識…）
- **機能語** … 品詞の種類（代名詞/前置詞/接続詞/限定詞/助動詞/疑問詞）
- **機能別フレーズ** … CEFR の言語機能（Functions）ごとの定型表現

> データ本体は `英単語_品詞別_意味グループ.xlsx`（各品詞＝1シート）と `data/*.csv`。

---

## 1. 動詞（Verbs）

「何を伝えたいか」を軸に12グループへ。基本動詞ほど句動詞（phrasal verbs）が多いので
コロケーション列を重視する。

| 意味グループ | 何を言いたいか | 代表語 |
|---|---|---|
| 思考・意見 | 考え・確信度を述べる | think, believe, guess, suppose, wonder, doubt |
| 伝達・発話 | 言う・尋ねる・説明する | say, tell, ask, speak, explain, mention |
| 依頼・提案・許可 | 頼む・勧める・許す | suggest, recommend, offer, invite, allow, let, help |
| 移動・動作 | 行く・来る・運ぶ | go, come, move, bring, take, send, reach |
| 変化・状態変化 | 〜になる・増減する | become, change, grow, increase, turn, rise, fall |
| 授受・所有 | あげる・もらう・持つ | give, have, hold, provide, pay, buy, spend |
| 知覚・感覚 | 見える・聞こえる・感じる | see, look, hear, feel, notice, seem |
| 感情・態度 | 好き・欲しい・心配 | like, love, want, need, enjoy, worry, prefer |
| 使役・産出・作成 | 作る・使う・させる | make, do, create, build, use, write, teach |
| 存在・発生 | ある・起こる・住む | be, happen, live, stay, remain, include |
| 認知・学習 | 知る・理解する・決める | know, understand, remember, decide, choose, find |
| 開始・継続・完了 | 始める・続ける・終える | begin, start, finish, try, keep, stop, end |

*接地*: CEFR Companion Volume の機能分類（意見表明・示唆(suasion)・確認など）＋
Beth Levin *English Verb Classes and Alternations* の意味クラス（motion, change of state,
communication, transfer, perception）。

## 2. 名詞（Nouns）

トピックで激しく入れ替わるが、全ジャンル横断の超高頻度語を意味フィールドで10グループに。

| 意味グループ | 代表語 |
|---|---|
| 時間 | time, year, day, week, hour, minute, moment |
| 人・関係 | people, friend, family, parent, teacher, guy |
| 場所・空間 | world, country, place, home, room, city, office |
| 抽象・プレースホルダー | way, thing, part, case, point, fact, kind, reason |
| 仕事・ビジネス | work, company, business, money, job, service, team |
| 社会・組織 | state, government, group, community, law, system |
| 情報・言語 | question, number, story, book, word, information |
| 身体・健康 | hand, eye, head, body, face, health, life |
| 出来事・活動 | problem, issue, game, change, result, end |
| 概念・知識 | level, power, right, back |

*接地*: 参考情報（COCA/NGSL トップ100名詞）。`thing / way / part / point` などは
具体物でなく状況・関係を包括的に指す**プレースホルダー**として高頻度。

## 3. 形容詞（Adjectives）

Dixon の意味型に沿って11グループ。修飾順序（VALUE→DIMENSION→AGE→COLOUR…）の
理解にもそのまま使える。

| 意味グループ（Dixon型） | 代表語 |
|---|---|
| 寸法・大きさ (DIMENSION) | big, small, large, long, short, tall, high, low, wide |
| 年齢・新旧 (AGE) | new, old, young, recent, modern |
| 評価・良し悪し (VALUE) | good, bad, great, nice, important, right, best, real, main |
| 色 (COLOUR) | black, white, red, blue, green |
| 物理的性質 (PHYSICAL PROPERTY) | hot, cold, warm, hard, soft, heavy, strong, clean, full |
| 人の性向・感情 (HUMAN PROPENSITY) | happy, sad, angry, kind, afraid, sorry, sure, busy, free |
| 速さ (SPEED) | fast, quick, slow |
| 難易 (DIFFICULTY) | easy, difficult, simple |
| 類似・関係 (SIMILARITY) | same, different, similar |
| 数量 (QUANTIFICATION) | many, much, few, whole, enough |
| 位置・順序 (POSITION/ORDER) | first, last, next, final |

*接地*: R.M.W. Dixon (1982; Dixon & Aikhenvald 2004) の形容詞意味型（13カテゴリ）を
コア語向けに再編。

## 4. 副詞（Adverbs）

文のニュアンス・論理を決める。意味的分類で8グループ。

| 意味グループ | 代表語 |
|---|---|
| 否定・焦点 | not, only, even, also, too, just |
| 程度・強調 | very, so, really, quite, almost, pretty, hardly |
| 頻度 | always, usually, often, sometimes, never, again |
| 時 | now, then, today, soon, already, still, yet, ago, later |
| 場所・方向 | here, there, back, away, home, out |
| 様態 | well, hard, carefully, quickly, slowly, together |
| 談話標識・接続 | however, therefore, actually, of course, though, anyway |
| 確信・態度 | maybe, probably, certainly, definitely, obviously |

*接地*: 参考情報（COCA トップ副詞の意味的分類）。談話標識（discourse markers）は
会話の流れを作るため独立グループにした。

## 5. 機能語（Function words）

内容は少数だが出現頻度は最上位。英作文の骨組み。種類別に6グループ。

| 意味グループ | 代表語 |
|---|---|
| 代名詞 | I, you, he, she, it, we, they, this, that, someone |
| 前置詞 | in, on, at, to, for, with, of, from, by, about, between |
| 接続詞 | and, but, or, so, because, if, when, while, although, until |
| 限定詞・冠詞 | the, a/an, some, any, every, no, all, both |
| 助動詞・法助動詞 | can, could, will, would, should, may, might, must, have to |
| 疑問詞 | what, who, where, when, why, how, which, whose |

*接地*: Nation の BNC/COCA 分析では機能語は語種は少ないが頻度最上位。
法助動詞（modals）は許可・依頼・丁寧度に直結するため、メモに丁寧度の使い分けを付す。

---

## 6. 機能別フレーズ（Functional exponents）

「品詞」ではなく**言語機能**で引くシート。会話でそのまま使える型を丁寧度別に用意。

| 機能 | 例 |
|---|---|
| 意見を言う | I think… / In my opinion,… / I'd say… |
| 同意・不同意 | I agree. / Good point. / I see your point, but… |
| 提案する | Why don't we…? / How about + -ing? / Let's… |
| 依頼する | Can you…? / Could you…? / Would you mind + -ing? |
| 申し出・招待 | Would you like…? / Shall I…? / Let me… |
| 許可を求める | Can I…? / Do you mind if I…? / May I…? |
| 確認・聞き返す | Sorry? / Do you mean…? / Could you say that again? |
| つなぎ(filler) | Well,… / Let me think. / Actually,… |
| 社交 | Nice to meet you. / How's it going? / Take care. |
| 談話の構造化 | First of all,… / On the other hand,… / To sum up,… |

*接地*: van Ek & Trim / CEFR Companion Volume の機能（Functions）分類。同一機能でも
丁寧度（カジュアル／標準／丁寧／フォーマル）で表現が変わる点を列で明示。

## 拡張ルール（Phase 2 に向けて）

1. 新語は上のいずれかの**意味グループに必ず割り当てる**（未分類を作らない）。
2. サブグループは既存に寄せる。無ければ新設し、この表にも1行追記する。
3. 1語は原則**1グループ1行**（多義で割れる場合は主用法を採用、他用法はメモへ）。
4. NGSL コア（約800〜1000語）まで、同じ列で `data/*.csv` に行追記 → `build_xlsx.py` を再実行。
