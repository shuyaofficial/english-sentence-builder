# 英文組み立て「母語思考回路ビルダー」

`06_単語管理` の単語DB（454語＋機能別フレーズ50）を素材に、**ネイティブの思考順
＝だれが → どうする（動詞）→ なにを** で英文をタイル操作で組み立てるアプリ。
動詞を選ぶと **文型（5文型）** が次のスロットを決める。依存ゼロ・オフラインで動く。

## ▶ 開き方

- **Web公開版**：<https://shuyaofficial.github.io/english-sentence-builder/>（mainへのpushで自動デプロイ）
- **ローカル**：`index.html` をブラウザ（Chrome推奨）で開くだけ。インストール不要・通信なし。
  （旧 `英文組み立て.html` はリダイレクトのスタブとして残してある）

## 使い方

- **主語スタート**：だれが → どうする → なにを/どんな → 味付け の順にタイルを選ぶ。
  make / find / keep / call / get などは**「文型」ボタンで5文型を切替**できる
  （例: make → SVO「〜を作る」/ SVOC「〜を…にする」）。**「進行」ボタンで be + 〜ing** にもできる。
- **機能スタート**：言いたいこと（意見/依頼/提案…）→ 型（Could you ___? 等）→ 中身をSVOで埋める。
- **必須フレーズ**：高レバレッジ30フレーズのチャンク学習（3つのサブタブ）。
  - *一覧*：カテゴリ別に眺める＋🔊。
  - *暗記カード*：日→英フラッシュカード。「言えた/もう一度」の自己評価を
    Leitner式SRS（0/1/3/7/14日間隔）として localStorage に保存し、次回の出題順に反映。
  - *穴埋め*：「Do you speak 〇〇?」等の 〇〇 に候補や単語DBの語を入れて自分の文にして🔊。
    できた文は **★ 文集に保存** できる。
  - *文集*：穴埋めで保存した自分の文を一覧・🔊・削除・すべてコピー（`srb.mysentences.v1`）。
- **逆クイズ**：日本語の文が出題 → 英語を組み立てて「採点」。模範解答を🔊で確認。
  出題ソース（動詞の例文/機能フレーズ/必須フレーズ/すべて）を選べて、
  **苦手・未出題の問題が優先的に出る**（正答履歴は localStorage `srb.quiz.v1`）。
- **🔀 日本語順で見る**：組んだ英文を日本語順（動詞が最後）に並べ替え、母語との違いを可視化。
- **キーボード**：↑↓で候補移動・Enterで確定・Backspaceで1つ戻す。検索窓は日本語でOK（例「依頼」）。
  必須フレーズ内は Space=カードをめくる・1/2=自己評価・←→=穴埋めのフレーズ切替。
- **🔊 読み上げ**：Web Speech API で発音（en-US系の音声を自動選択）。

## 文法サポート（動詞が文型を決める）

| 動詞の型 | 次に必要なもの | 例 |
|---|---|---|
| SV（自動詞） | なし（副詞・場所は任意） | I go. |
| SVC | 補語（形容詞/名詞） | It is **big**. / I am **a student**. |
| SVO | 目的語（名詞） | I like **coffee**. |
| SVOO | 人 ＋ 物 | I gave **her a book**. |
| SVOC | 目的語 ＋ 補語 | It makes **me happy**. |

主語が he/she/it なら3単現の -s、否定は don't/doesn't/didn't ＋原形、冠詞 a/an の要否などを
組み立て中に「文法スポット」として表示する。

## 構成（このフォルダ）

```
web/
├── index.html               # エントリ（構造のみ。旧名 英文組み立て.html はリダイレクト）
├── app.css                  # スタイル（DESIGN.md トークン準拠）
├── grammar.js               # 活用/文型/進行形/冠詞/文法スポットの純ロジック
├── speech.js                # 読み上げ（en音声の優先選択）
├── phrases.js               # 必須フレーズモード（一覧/暗記カードSRS/穴埋め/文集）
├── quiz.js                  # 逆クイズの出題プール・苦手優先・正答履歴
├── app.js                   # state＋描画＋イベント（モード/文型・進行切替/並べ替え）
├── sw.js                    # Service Worker（オフラインキャッシュ）
├── manifest.webmanifest     # PWAマニフェスト
├── icons/                   # アプリアイコン
└── sentence_builder_data.js # 生成物（直接編集しない）
```

## 📱 スマホにインストール（PWA）

公開URLを開いて、iOS Safari なら「共有 → ホーム画面に追加」、Android Chrome なら
「メニュー → アプリをインストール」。以後は**アプリとして起動でき、圏外でも動く**。
テストは `node tests/node_smoke.js` と `node tests/mine.test.js`（依存ゼロ・node のみ）。

学習の進み具合（SRSの箱・期限）はブラウザの localStorage（キー `srb.phrases.v1`）に
保存される。消したいときはブラウザのサイトデータ削除でリセットできる。

## データを増やす・作り直す

語彙は `../data/*.csv` が編集元。CSVに行を足したら：

```bash
python3 ../build/build_web.py      # data/*.csv → sentence_builder_data.js を再生成
```

文型の既定・分岐は `build_web.py` の `PATTERN_PRIMARY` / `PATTERN_ALLOWED` で調整できる。
