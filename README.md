# 06_単語管理 — 品詞別 × 意味グループ 単語データベース

英会話で**言いたいことをスムーズに英作文**するための、**逆引き**できる語彙集。
コア語彙を「品詞 → 意味グループ → サブグループ」に細分化し、日本語の
**トリガー（言いたいこと）**から必要な語・表現へ最短で到達できる。

## 🧭 まずここから

1. **▶ 組み立てアプリ（Web公開版）**：<https://shuyaofficial.github.io/english-sentence-builder/>
   — 5文型×進行形の組み立て・必須フレーズSRS・逆クイズ（苦手優先）・PWA対応で
   スマホのホーム画面に追加可。ローカルは [web/index.html](web/index.html)（[説明](web/README.md)）
2. **メイン（一覧）**：[`英単語_品詞別_意味グループ.xlsx`](英単語_品詞別_意味グループ.xlsx) を開く
3. **分類の全体像と根拠**：[分類表.md](分類表.md)
4. **拡張・再生成**：`data/*.csv` を編集 → `python3 build/build_web.py`（XLSXは `build/build_xlsx.py`）。
   main に push すると GitHub Actions が `web/` を自動デプロイする

## 🔎 使い方（逆引き 3ステップ）

1. 品詞タブを選ぶ（**動詞／名詞／形容詞／副詞／機能語／機能別フレーズ**）
2. ヘッダーの ▼フィルタで **意味グループ**を絞る（例: 動詞→「思考・意見」）
   - または **`言いたいこと`列を Ctrl+F** で日本語検索（例:「依頼」「〜になる」「たぶん」）
3. `見出し語`・`コロケーション・句動詞`・`例文` を見て、そのまま口に出す／書く

> 逆引きキー（`言いたいこと` / `機能`）列は**スクロールしても左端に固定**表示される。

## 🗂 データ辞書（内容語シートの列）

| 列 | 内容 |
|---|---|
| `言いたいこと` | 逆引きキー。日本語の発話意図 |
| `意味グループ` / `サブグループ` | 何を伝えたいかによる分類（できる限り細分化） |
| `見出し語` | 英単語（lemma） |
| `コア語義` | 中心的な日本語義 |
| `活用・派生` | 動詞:過去/過去分詞/-ing、形容詞:比較級/最上級、名詞:不規則複数 など |
| `コロケーション・句動詞` | よく共起する語・句動詞（会話で塊で使う） |
| `例文(EN)` / `例文(JP)` | 短く自然な会話例 |
| `頻度ランク` | COCA/NGSL 由来の**概算**順位（`100+`＝概略、機能語は英語全体順位） |
| `レジスター` | 日常／口語／ビジネス／書き言葉 |
| `メモ` | ニュアンス・使い分け・紛らわしい語 |

機能別フレーズシートは `機能 / 丁寧度・場面 / トリガー(日) / 表現(EN) / 意味(JP) / 例文 / メモ`。

```
06_単語管理/
├── README.md                       # このハブ

├── 英単語_品詞別_意味グループ.xlsx    # メイン成果物（品詞ごとにシート）

    ├── seed_data.py                # data/*.csv を初期生成
```

**設計思想**：CSV が唯一の編集元。語彙追加＝CSVに行追記 → `build_xlsx.py` 再実行で完了。
差分がテキストで追え、将来 `build_html.py`（同じCSV＋`../DESIGN.md`トークン）でHTML化もできる。
